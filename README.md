## Quick Start

```bash
git clone https://github.com/Nehareddy0404/context-diff.git
cd context-diff
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) — run tests with `npm test`

## What This Is

Your AI forgets you after every conversation. Context Diff is a prototype that fixes that by treating personal context like source code — you can version it, branch it, diff it, and search it.

## Features

| View | What it does |
|------|-------------|
| 📜 **Timeline** | Commit-log style history of your context snapshots |
| 📊 **Diff View** | Compare any two snapshots with color-coded changes |
| 🌿 **Branches** | Branch graph + automatic conflict detection |
| 📋 **Context** | Current state with semantic blame and permission levels |
| ⚡ **TTFT** | Dashboard showing 87% reduction in time to first token |
| 🔍 **Search** | Full-text search across all branches and history |
| 🌗 **Themes** | Light and dark mode with one-click toggle |

### What Makes It Interesting

- **Semantic Blame** — trace when, why, and by whom each context entry changed
- **Conflict Detection** — spots contradictions between branches automatically
- **Permission Levels** — control what context is shared vs. private across branches
